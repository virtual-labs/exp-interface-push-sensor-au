Raspberry Pi Button/IR/LDR LED Control

Aim : To interface LED with Raspberry Pi and write a program to turn ON LED or 1 sec after every 2 seconds.

Instructions

  • Click on the components to start interacting it with in the connection area
  • Hover over the rasberrypi connections to get informations about various connections pin
  • Double click on various connections points to make connections
  • Double click on the empty space in the connection area to extend and bend wire connections

Procedure

  1. In Experiment 3, connect any GPIO pin to any of the pins of the button.
  2. Connect the other pin of the button to any of the GND pins of the Rasberry Pi.
  3. Make a connection between Rasberry Pi and Led's anode pin via any of the GPIO pins.
    • Double-click on any of the GPIO pins to start wiring.
    • Make a clear path till the anode of the Led pin.
    • Double-click on the anode of the led pin to complete the connection
  4. The pin requires low voltage so put a resistor between the Rasberry Pi and Led
    • Double-click on the cathode to start wiring between the resistor and led pin
    • Choose any of the resistor pins to complete the wiring
  5. Connect the other resistor's pin to any of the GND pins of Rasberry Pi
  6. Enter the GPIO pin number of the button and led to the code section. The code section can be opened by clicking the Code button on the right upper side of the simulation area.
